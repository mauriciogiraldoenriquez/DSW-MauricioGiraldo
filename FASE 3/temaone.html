Etiquetas Básicas de html:

HTML es un lenguaje usado para crear documentos y páginas web. Este lenguaje utiliza unas etiquetas especiales intercaladas en un documento de texto sin formato. Estas etiquetas son interpretadas posteriormente por los exploradores encargados de visualizar la página o el documento con el fin de establecer el formato.

Como lo dice Ceballos (2015)” las etiquetas indican a los explorares web cómo tienen que mostrar el texto y los gráficos” (p. 441). Estas generalmente se escriben entre los símbolos < y >, también suele haber una etiqueta de comienzo y una de fin como, por ejemplo: (< texto etiqueta>) y (/) respectivamente, que enmarcan el texto que va a ser formateado por ellas. Muchas etiquetas incluyen distintos atributos que detallan la forma de mostrar el texto que aparece entre ellas.

La etiqueta de apertura incluye el atributo tamaño cuyo valor asignado es 10, luego aparece el texto que se desea mostrar con el tamaño especificado y la etiqueta de cierre. Así mismo las etiquetas o marcas permiten incluir elementos no contenidos en el texto como imágenes, crear hiperenlaces, agregar significado al texto, etc.
Atributos y valores
Los atributos les aportan información específica a las etiquetas. Se colocan junto a las etiquetas de inicio actuando junto con ellas. Cualquier etiqueta puede presentar de cero a más atributos separados por espacios.

Por ejemplo:

body bg color = “blue”
En el ejemplo, se tiene el atributo bg color con el valor blue. Este código forma un elemento, el cual es una estructura mediante la cual se organiza el contenido del documento o acciones que tienen lugar cuando el navegador lo interpreta. Los elementos están formados por la etiqueta de inicio, la etiqueta de fin y lo que se encuentra entre ambas etiquetas.
Valor de un atributo:
Cuando se utiliza un atributo HTML se debe especificar su valor y este puede representar una cantidad exacta, característica específica o en qué mediada se ejecutará dicho atributo. El valor se expresa entre comillas luego del signo igual (=). De acuerdo al atributo puede haber una variedad de valores disponibles.
Por ejemplo:
En el caso de align (alinear) Puede tomar el valor de: Center (centro), right (derecha), left (izquierda), (justify) justificado. El atributo align, con su valor en right aplicado a un párrafo hará que el texto este alineado a la derecha.
