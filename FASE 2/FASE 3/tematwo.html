Atributos y valores

Los atributos les aportan información específica a las etiquetas. Se colocan junto a las etiquetas de inicio actuando junto con ellas. Cualquier etiqueta puede presentar de cero a más atributos separados por espacios.

Por ejemplo:

< body bgcolor= “blue”>

En el ejemplo, se tiene el atributo bgcolor con el valor blue. Este código forma un elemento, el cual es una estructura mediante la cual se organiza el contenido del documento o acciones que tienen lugar cuando el navegador lo interpreta. Los elementos están formados por la etiqueta de inicio, la etiqueta de fin y lo que se encuentra entre ambas etiquetas.

Valor de un atributo:

Cuando se utiliza un atributo HTML se debe especificar su valor y este puede representar una cantidad exacta, característica específica o en qué mediada se ejecutará dicho atributo.

El valor se expresa entre comillas luego del signo igual (=). De acuerdo al atributo puede haber una variedad de valores disponibles.

Por ejemplo: En el caso de align (alinear) Puede tomar el valor de: Center (centro), right (derecha), left (izquierda), (justify) justificado. El atributo align, con su valor en right aplicado a un párrafo hará que el texto este alineado a la derecha. < p align="right" >Este es un párrafo alineado a la derecha < /p >
