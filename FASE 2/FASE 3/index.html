El tema que vamos a tratar son las etiquetas básicas HTML y para esto es bueno saber qué es el lenguaje HTML.

Este es un lenguaje utilizado para desarrollar páginas y documentos web y este, a diferencia de los lenguajes convencionales utiliza etiquetas especiales intercaladas en un documento de texto sin formato, las cuales después serán interpretadas por los exploradores encargados de visualizar la página o documento web para poder establecer el formato.

HTML basa su sintaxis en la etiqueta, esta es la instrucción y se compone frecuentemente de dos partes: una apertura de forma general < etiqueta > y un cierre de tipo < / etiqueta >

Estas etiquetas pueden incluir atributos, que son elementos de las etiquetas que pueden proporcionar ciertas características a un texto, imagen, etc. Por ejemplo: Color, altura, ancho, alineación.
